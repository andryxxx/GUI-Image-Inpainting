# GUI-Image-Inpainting

This is a simple GUI for [inpaint filter](http://scikit-image.org/docs/dev/auto_examples/filters/plot_inpaint.html) of scikit-image 

## Running

* `git clone <repository-url>` this repository
* `pip install -r requirements.txt`
* `python imageprocessor.py`

## Usage
* Click on Upload
* Select the image you want
* Select a rectangular area in the image
* Click Trasform

The edited image will be save as "name of file"+**_out**+"extension" in the directory of the original image
